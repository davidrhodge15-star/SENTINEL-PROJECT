# 🛡️ Sentinel: Advanced Diagnostics Engine

A high-performance system built to process raw data streams and provide real-time diagnostic analytics.

## 🛠️ The Tech Stack
* **Frontend:** React.js, Tailwind CSS, Lucide-React
* **Data Visualization:** Recharts
* **Backend:** Python, FastAPI, Pandas
* **Export:** jsPDF / jspdf-autotable

## 🌟 Key Features
* **Data Processing Engine:** Custom Python backend using Pandas to sanitize and categorize complex datasets.
* **Dynamic Analytics:** Real-time bar chart visualization of data distribution.
* **Dynamic UI:** Responsive dashboard with conditional formatting and live status tracking.
* **Professional Export:** Generates formatted PDF reports directly from the browser.

## 🚀 How to Run

### 1. Start the Backend
```bash
cd backend
source venv/bin/activate
python main.py
```

### 2. Start the Frontend
```bash
cd frontend
npm run dev
```
